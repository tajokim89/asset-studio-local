# GPT Image 2 community walk reference

- Source: https://community.openai.com/t/developing-sprite-sheets-with-gpt-image-2/1379831
- Retrieved: 2026-07-12
- `mannequin_walk_12f_source.jpg`: original 12-frame mannequin sheet attached to the thread.
- `poses/mannequin_walk_01.jpg` through `poses/mannequin_walk_12.jpg`: individual pose crops in source order.

These are side-profile walk poses, not SE/isometric poses. They are retained to test whether GPT Image 2 can transfer explicit volumetric pose geometry from Image 1 while preserving character identity from Image 2.
