# Development History

이 디렉터리는 과거 개발 마일스톤별 계획, 구현 보고서, QA 결과를 보존하는 아카이브입니다.

- `milestones/`: 완료된 구현·핫픽스 보고서
- `plans/`: 완료된 작업 계획과 계약 문서
- `artifacts/`: 당시 생성·검증 결과 메타데이터

여기의 번호는 작업이 진행된 순서를 기록하기 위한 역사적 표식입니다. 현재 제품 버전, 런타임 기능 플래그, CSS 클래스, JavaScript 캐시 키로 사용하지 않습니다.

현재 사용법과 기능은 프로젝트 루트의 `README.md`를 기준으로 확인합니다.
